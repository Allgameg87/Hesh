Задание по хэшированию

Написать программу на Python, которая вычисляет хэш-значения SHA-256 для заданного текста.

Решение на Python:

import hashlib

def calculate_sha256(text):
    # Создаем объект хэша SHA-256
    sha256_hash = hashlib.sha256()
    
    # Обновляем хэш с нашим текстом (предварительно кодируя в bytes)
    sha256_hash.update(text.encode('utf-8'))
    
    # Получаем шестнадцатеричное представление хэша
    return sha256_hash.hexdigest()

# Вычисляем хэш для строки "test"
text_to_hash = "test"
hashed_value = calculate_sha256(text_to_hash)

print(f"Текст: '{text_to_hash}'")
print(f"SHA-256 хэш: {hashed_value}")

Результат выполнения:

Текст: 'test'
SHA-256 хэш: 9f86d081884c7d659a2feaa0c55ad015a3bf4f1b2b0b822cd15d6c15b0f00a08

Проверка через командную строку

Windows:

echo|set /p="test" > %TMP%/hash.txt | certutil -hashfile %TMP%/hash.txt SHA256 | findstr /v "hash"

Результат:

9f86d081884c7d659a2feaa0c55ad015a3bf4f1b2b0b822cd15d6c15b0f00a08

Linux:

echo -n test | sha256sum

Результат:
